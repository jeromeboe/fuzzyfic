1) Définir les variables linguistiques :
	- Nom : Qualité du service
	- Range value : [1,10]
	- Sous-ensembles flous : {mauvais, bon, excellent} avec leurs fonctions d'appartenance

2) Définir les opérateurs logique, ET, OU

3) Définir les régles :
	- Si le service est mauvais ou la nourriture est exécrable => le pourboire est faible.
	- Si le service est bon => le pourboire est moyen.
	- Si le service est excellent ou la nourriture est délicieuse => le pourboire est élevé.


4) Fuzzificaion pour des inputs donnés.

5) Défuzzifier

=====

Example :

# Variable input 1
service_quality[:mauvais] = Fuzzyfic::Trapezoid.new([0,4], [0,3])
service_quality[:bon] = Fuzzyfic::Trapezoid.new([3,7], [4,6])
service_quality[:excellent] = Fuzzyfic::Trapezoid.new([6,10], [7,9])

# Variable input 2
nourriture[:execrable] = Fuzzyfic::Trapezoid.new([0,4], [0,3])
nourriture[:delicieuse] = Fuzzyfic::Trapezoid.new([0,4], [0,3])

# Variable output
tips[:faible] = Fuzzyfic::Triangle.new([0,10], 5)
tips[:moyen] = Fuzzyfic::Triangle.new([10,20], 15)
tips[:eleve] = Fuzzyfic::Triangle.new([20,30], 25)


# Rules
rule1 = service_quality[:mauvais].or(nourriture[:execrable]).then tips[:faible]
rule2 = service_quality[:bon].then tips[:moyen]
rule3 = service_quality[:excellent].or(nourriture[:delicieuse]).then tips[:eleve]


# ApplyRules
rule1.apply(7.83, 7.32)
